---
description: ProjectSpecs archive workflow
---

Run `/ps:archive` to snapshot a completed change.

Required input (ask if missing): change name.

Read:
- `projectspec/changes/<change>/delivery.md`
- Related specs and ADRs
- `projectspec/mapping/traceability.yaml`

Write:
- `projectspec/archive/<change>/`

Include the delivery plan and traceability snapshot.
